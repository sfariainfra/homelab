# My Linux Homelab

A hands-on Linux infrastructure project showcasing system administration, virtualization, networking, monitoring, automation, and self-hosted services.

## Overview

This repository documents my personal homelab environment, built to develop practical skills in Linux administration, virtualization, networking, monitoring, automation, and self-hosting.

## Current Environment

- Fedora Server
- KVM / Libvirt
- Docker
- Nginx
- Tailscale
- Samba
- Jellyfin
- Prometheus
- Grafana
- Loki
- Alloy
- N8N

## Goals

This homelab was built to gain hands-on experience with Linux system administration, networking, virtualization, monitoring, automation, and self-hosted services.

The environment serves as a learning platform where I can deploy, maintain, troubleshoot, and document real-world infrastructure scenarios.
